How it all began?

With various motivations, me and a few other students signed up for the SDA Academy course, where we wanted to learn the Python programming language from scratch. This should bring us closer to the opportunity to get a job as a Junior Python Developer.

  • Python - basics

    It really started from the beginning. We went through ones and zeros to the first computers. Several of us wrote the first line of code. Yippee! I gradually repeated the complete basics of Python, from variables and functions to conditions and loops.

    November 2020

  • Software Testing and TDD

    Software development also includes testing it. We explained the importance of testing and tried to write our own tests. No matter how great a programmer you are - you will always make mistakes.

    December 2020

  • Python - intermediate

    Some of us have discovered object-oriented programming and started asking: "What the hell is that?" It's a way to think about code and write it. Classes, instances, attributes and methods are no longer foreign words to us. Then came the decorators, generators, iterators, and we were ... yes, right there.

    January 2021

  • Algorithms and data structures

    As with the recipe for cooking: it must be clear from each algorithm notation exactly what the computer should do. Recursion will scare me in a dream for a long time to come and probably I will never forget the sorting algorithm dance scenes.

    February 2021

  • Design patterns and good practices

    IMPORT THIS. That's the main thing I need to know.

    February 2021

  • SQL databases and their programming

    A SQL query goes into a bar, walks up to two tables and asks: "Can I join you?"

    March 2021

  • HTML, CSS and JavaScript

    One for all, all for one. Like three musketeers kind and friendly. Elegant with a refined style. What a wonderful world this Frontend is.

    April 2021

  • Backend technologies

    Kitchen in full operation. I have learned about the raw materials. Now, I will learn to cook. With this unchained Django, it's a really wild ride.

    May 2021

  • Final project

    In the end, my study efforts make sense and tangible results. Let's try my app and enjoy it!
